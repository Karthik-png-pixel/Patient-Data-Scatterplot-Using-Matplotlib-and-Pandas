# Project Overview: Imported patient public health datasets consisting of both male and female weights and height. Created new data frames using pandas to gain statistical insights into the data such as minimum and median heights and weights amidst a scatterplot using matplotlib packages. Labeling of the scatterplot including titles, patient of interest, and axis scale were performed using the scatterplot visualization, which indicated a generally positive correlation was observed between height and weight in male patients.

# Tools used: Python, Pandas, Matplotlib
